# Segment Tree

* Able to
 * Answer min/max query within interval.
 * Update within fixed-size array.
* Variant
 * Answer sum query within interval (would rather use BITree).
 
## Example

```
           [0, 9)
       /            \
    [0, 4)          [4, 9)
    /    \          /    \
 [0, 2)  [2, 4)  [4, 6) [6, 9)
             ...
```
